# ⚡ Voltix E-Commerce Website  

A modern and fully responsive **E-Commerce web application** built with **React.js**, offering a seamless shopping experience with cart management, user authentication, and a payment system.  

---

## 🚀 Features  

### 👤 User Authentication  
- Signup & Login system with validation.  
- User data stored in **Local Storage**.  
- Restricted access: only logged-in users can shop or checkout.  

### 🛒 Shopping Cart  
- Add products from multiple categories.  
- Update product quantities or remove items.  
- Cart summary with total price calculation.  
- Clear entire cart in one click.  

### 💳 Payment System  
- Billing form with **regex validation** (Full name, email, card details, etc.).  
- Multiple payment options: **Card, PayPal, Cash on Delivery**.  
- Confirmation screen with success message + redirect to Home.  

### 📦 Product Categories  
- Smart Phones 📱  
- Smart Watches ⌚  
- Audio Devices 🎧  
- Cameras 📸  
- Laptops 💻  
- Setups 🖥️  

### 🎨 UI/UX  
- Built with **Bootstrap 5** and custom CSS.  
- Sticky responsive Navbar with search functionality.  
- FontAwesome icons for modern styling.  
- Mobile-first responsive design.  

### 🔀 Routing  
- Smooth navigation using **React Router v6**.  
- Custom **404 Not Found** page.  

---

## 🛠️ Tech Stack  

- **Frontend**: React.js  
- **UI Framework**: Bootstrap 5, CSS3  
- **Routing**: React Router v6  
- **State Management**: React Context API  
- **Storage**: Local Storage  
- **Icons**: FontAwesome
